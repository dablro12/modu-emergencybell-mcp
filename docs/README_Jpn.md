## これは何ですか？

韓国で外出中に急なとき、**119 と 1339 の違い**、**夜間の薬局**、**近くのトイレ**、**夜道の安全ベル** など、わからないことが多いです。

**みんなの非常ベル（modu-emergencybell）** は KakaoTalk のエージェントと連携し、**日本語で質問**するだけで、近くの施設・電話番号・次に取るべき行動を案内します。

> **情報提供のみ**です。電話の代行・診断・通報の代行はしません。

## 使い方

### KakaoTalk · PlayMCP ユーザー

1. PlayMCP で **みんなの非常ベル** MCP を接続します。
2. **場所＋状況** を一文で送ります（**日本語でOK**）。
3. エージェントが近くの施設・緊急電話・次の行動を返します。

**ヒント**
- 「江南駅」「明洞」「梨泰院」など **おおよその場所** で十分です。
- 回答の **韓国語住所はそのまま** — タクシーや店員に見せやすいです。
- エージェントが回答を **日本語に翻訳** できます。

## こんな方のために

- 🧑‍🤝‍🧑 **住民** — トイレ・診療所・薬局・救急の情報  
- 🌏 **観光客** — **日本語・英語・中国語** で質問可能  
- ♿ **障がい・介護** — 車椅子トイレ、ベビー・おむつ交換  
- 🌙 **夜の安全** — 安全ベル、子ども・青少年のシェルター  
- 🎖️ **功労者** — 委託病院の案内  

---

## ご注意

- 命の危険があるときは **すぐ 119** に電話してください。
- 夜間・休日の病院・薬局は **曜日単位** の情報です。深夜営業は **電話確認** をおすすめします。
- 案内は **参考情報** であり、医療診断や緊急出動の代行ではありません。
