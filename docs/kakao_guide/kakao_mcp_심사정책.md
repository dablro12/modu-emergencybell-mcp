## 기본 심사 정책

- MCP Server는 최소 1개 이상의 Tool을 포함해야 합니다.
- 1개의 MCP Server가 너무 많은 Tool을 보유한 경우 AI가 올바른 툴 선택을 하기 어렵습니다. 적정 Tool 개수는 3~20개를 권장합니다.
- 심사 완료 후에도 정책 위반이 감지되면 관리자 판단에 따라 Tool이 비활성화되거나 등록이 철회될 수 있습니다. 이 경우 사유는 신청자에게 개별 안내됩니다.
- 등록한 MCP가 표준 스펙에 위배되거나 비정상적인 동작이 감지되는 경우 반려 사유가 될 수 있습니다.
- 자사 서비스가 아닌 제3의 서비스 인증 정보를 사용하는 경우 등록자와 인증 정보사의 관계에 따라 심사가 반려될 수 있습니다.
- MCP Server를 자동 생성하는 제3자 플랫폼(ex. https://zapier.com/mcp)에서 생성된 MCP는 명확한 소유자 검증 및 정책 준수가 어려워 등록이 허용되지 않습니다.
- 동일 기능 구조의 MCP가 반복적으로 공개 MCP 등록 심사를 요청하는 경우, 등록이 제한될 수 있습니다.
    
    ※ 단순히 명칭, 문구, 출력 형식만 변경한 MCP는 별도의 신규 MCP로 인정되지 않습니다.
    
- LLM이 자체 웹 검색을 통해 충분히 구현 가능한 기능만을 제공하는 MCP는 심사 과정에서 반려 또는 공개 제한될 수 있습니다.
    
    ※ 단, LLM의 기본 기능을 확장하거나 기능을 추가하는 목적이 명확히 드러날 경우 예외적으로 허용
    
- 내부 MCP 품질 기준(안정성, 창의성, 응답 일관성 등)에 미달하는 경우, 관리자 판단에 따라 비공개 또는 개선을 요청할 수 있습니다.
- MCP의 이름과 설명은 기능을 명확히 전달해야 하며, 추상적이거나 모호한 표현이 포함된 경우 심사 중 수정 요청이 이루어질 수 있습니다.
- 등록한 MCP의 도구 실행에 유료 결제 계정 또는 유료 구독이 필수적으로 요구되는 경우 반려 사유가 될 수 있습니다.
- MCP가 제공하는 데이터의 출처가 명확하지 않은 경우 승인이 반려될 수 있으며, 운영자는 데이터의 출처 및 구성 확인을 위해 증빙 자료를 요청할 수 있습니다.
- 제3자로부터 데이터 무단 사용 제보가 접수되거나 그 사실이 확인된 경우, 운영자는 사전 통보 없이 해당 MCP를 비공개 처리할 수 있습니다. 이 경우 등록자는 데이터 사용 권한에 대한 증빙(소속 확인, 사용 허가 등)을 제출해야 하며, 증빙이 확인되기 전까지 비공개 상태가 유지됩니다.

## Tool의 동작

- 등록된 모든 Tool은 실제 동작 여부를 반드시 사전 테스트 해야 하며, 오류 없이 응답이 오는지 확인해야 합니다.
- 응답이 지나치게 느리거나 timeout이 자주 발생하는 경우 반려 사유가 될 수 있습니다.
- Tool이 정상 동작하더라도, 과도한 리디렉션, 크롤링 지연, 불필요한 외부 호출이 감지될 경우 성능 저해로 반려될 수 있습니다.
- 필요한 경우 Tool의 답변에 아웃 링크가 포함되어도 좋습니다.
- 답변 내 상업적 링크, 구매 유도, 리워드 제공 등 명시적 또는 암묵적 상업 행위가 지나치게 포함될 경우 반려 사유가 될 수 있습니다.
- 답변에 비속어/정치적/성적인 내용이 포함되어 있거나 일반적인 사회 통념에 위배되는 경우 반려 사유가 될 수 있습니다.
- 응답에 보안상 위험이 있거나 악의적으로 이용될 수 있는 파일 다운로드를 포함하거나 유도할 경우, 심사가 반려되거나 MCP가 비공개 처리될 수 있습니다.
- Tool 의 Response가 24k를 초과하면 에러로 처리되기 때문에 반려 사유가 될 수 있습니다. Response 텍스트가 과도하게 크지 않도록 확인 부탁드립니다.

## Name / Description 규칙

- MCP Server Name 또는 Tool Name에 "kakao"를 prefix 또는 suffix 로 사용할 수 없습니다. 꼭 필요한 경우 사전 협의가 필요합니다.
- Tool Description은 LLM이 잘 이해할 수 있도록 자세하게 기술되어야 하며, 언어는 영어 또는 한국어를 권장합니다.
- Name은 간결하게, 서비스 기능을 대표할 수 있는 명칭이어야 하며, 'AI', 'Bot', 'Service' 등의 중복 키워드는 지양해 주세요

## 개인정보

- Tool의 기능과 무관한 개인정보를 수집하거나 요구해서는 안됩니다.
- 반드시 필요한 경우를 제외하고는 사용자의 인증 정보나 민감 정보를 외부로 전송하여서는 안됩니다.
OAuth 인증이나 Token / Key 입력 등을 통해 획득한 인증 정보는 Tool의 사용 목적 외에 다른 용도로 사용해서는 안됩니다.
- Tool이 아래 정보를 요구하거나 response로 전송하는 경우 심사가 반려될 수 있습니다.
    - 주민등록번호
    - 운전면허번호
    - 여권번호
    - 외국인 등록번호
    - 카드번호
    - 계좌번호

## 심사 진행 및 상태 안내

- 심사 요청한 MCP는 영업일 기준 7일 이내에 심사가 완료됩니다.(평균 1~2일)
- 심사 진행 상태에 대한 안내 메일은 "내 정보 관리 > 연결된 이메일"의 이메일 주소로 보내집니다.
- 심사 중 별도의 문의 또는 보완 요청이 발생할 수 있으며, "내 정보 관리 > 연결된 이메일"로 메일이 발송됩니다.
- 심사에 관한 추가 문의사항은 PlayMCP 디스코드를 이용해 주세요.
- 심사가 완료되어 게시 중인 MCP의 Tool 정보가 변경될 경우 반드시 MCP 정보 수정에서 “정보 불러오기”를 다시 시행하여 주셔야 정보가 갱신됩니다. (이 때 재심사가 진행됩니다.)

## 기타 사항

- 인증이 필요한 상황에 인증 정보가 없거나 만료된 경우 401(unauthorized) Http 응답이 필요합니다.
- 심사가 통과된 공개 MCP의 서버가 불안정하거나 지속적으로 응답이 없는 경우, 운영자가 MCP를 비활성화할 수 있습니다.
- MCP 대표 이미지에 움직이는 이미지는 사용할 수 없습니다.
- MCP의 대표 이미지는 서비스의 성격에 부합해야 하며, 저품질 또는 부적절한 이미지를 사용할 경우 심사에서 반려될 수 있습니다.
- PlayMCP는 현재 MCP 스펙의 Resource와 Prompt 정보는 다루지 않습니다.
